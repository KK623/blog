2. 研究背景

2.1 Factorization Machines (FM) 基础

Factorization Machines 是 Steffen Rendle 于 2010 年提出的经典模型，用于解决稀疏数据下的特征交互问题。其核心思想是将每个特征表示为一个低维向量，通过向量内积建模特征交互：

传统 FM 公式：
ŷ(x) = w₀ + Σ wᵢ xᵢ + Σ Σ ⟨vᵢ, vⱼ⟩ xᵢ xⱼ

其中 vᵢ 是特征 i 的隐向量，⟨vᵢ, vⱼ⟩ 表示向量内积，用于建模二阶特征交互。

2.2 现有 FM 变体的局限

(1) 高阶扩展困难：

  • 传统 FM 主要关注二阶交互，扩展到高阶时计算复杂度急剧上升
  • 高阶 FM (HoFM) 需要显式枚举所有高阶组合，参数爆炸
  • DeepFM、xDeepFM 等深度方法虽能隐式捕获高阶交互，但缺乏显式可解释性

(2) 固定交互模式：

  • 现有方法通常预定义最大交互阶数（如2阶、3阶）
  • 无法自适应地学习最优的交互阶数
  • 不同特征之间的关系被等同对待，缺乏关系感知能力

(3) 指数项爆炸问题：

  • 显式高阶交互需要计算 O(dᵏ) 项，k 为交互阶数
  • 即使采用近似方法，仍面临计算效率和模型容量的权衡

2.3 复数表示的启示

EulerNet 和 EulerFormer 的成功表明，复数向量空间为特征表示提供了新的可能性：

  • 复数乘法的角度相加特性天然适合建模交互关系
  • 模和相位的分离表示提供了更丰富的表达能力
  • 欧拉公式为特征变换提供了优雅的数学工具
核心问题： 如何将复数旋转机制与经典的 Factorization Machines 框架结合，实现无界阶、关系感知的特征交互学习？

3. 研究动机

3.1 从向量内积到复数旋转

传统 FM 使用向量内积 ⟨vᵢ, vⱼ⟩ 建模特征交互。作者思考：能否用复数旋转来替代内积运算？

关键洞察：

  • 内积运算本质上是角度和模的耦合计算
  • 复数旋转可以更清晰地分离角度信息和强度信息
  • 通过极坐标表示，可以将特征交互转化为角度相加问题

3.2 极角表示的优势

RFM 将每个特征表示为复数平面上的一个极角：

特征表示： f(θ) = e^(iθ) = cos(θ) + i·sin(θ)

这种表示的优势：

  • 角度编码关系： 两个特征的交互可以通过角度相加自然表示
  • 旋转不变性： 复数旋转保持了特征间的相对关系
  • 无界阶交互： 多个特征的交互转化为多个角度的累加，无阶数限制

3.3 关系感知的需求

不同特征对之间的关系强度应该是不同的。RFM 引入自注意力机制来学习这种关系：

  • 通过自注意力计算特征对之间的权重
  • 根据关系强度自适应地调整旋转角度
  • 实现关系感知的特征交互建模

4. 核心技术点

4.1 极坐标特征表示

RFM 的核心创新是将特征从传统的实数向量表示转换为复数平面上的极角表示。

表示方法：

  • 每个特征 field 对应一个旋转角度 θ
  • 特征值编码为复数单位向量：z = e^(iθ) = cos(θ) + i·sin(θ)
  • 模固定为1，信息完全编码在相位（角度）中

4.2 旋转特征交互

核心机制： 特征交互通过复数乘法（旋转）实现。

二阶交互：
zᵢ · zⱼ = e^(iθᵢ) · e^(iθⱼ) = e^(i(θᵢ+θⱼ))

高阶交互（无界阶）：
z₁ · z₂ · ... · zₖ = e^(i(θ₁+θ₂+...+θₖ))

优势：

  • 任意阶数的交互都可以通过角度累加计算
  • 计算复杂度为 O(k) 而非 O(dᵏ)
  • 无需预定义最大交互阶数

4.3 自注意力旋转函数

为了实现关系感知的交互，RFM 引入了自注意力机制：

注意力权重计算：

  • 计算特征对 (i, j) 之间的注意力分数 αᵢⱼ
  • αᵢⱼ 反映特征 i 和 j 之间的关联强度
  • 使用 softmax 归一化注意力权重

自适应旋转：

  • 根据注意力权重调整旋转角度
  • 重要特征对产生更大的旋转影响
  • 实现关系感知的特征交互

4.4 显隐式交互统一

RFM 统一建模了两种特征交互：

(1) 显式交互（基于旋转）：

  • 通过复数旋转显式计算特征交互
  • 可解释性强，知道哪些特征参与了交互
  • 支持任意高阶交互

(2) 隐式交互（基于神经网络）：

  • 通过 MLP 网络隐式学习复杂模式
  • 捕获非线性、高阶的隐式关系
  • 增强模型的表达能力

(3) 融合机制：

  • 显式交互提供明确的交互信号
  • 隐式交互补偿显式交互的局限
  • 两者相互增强，提升整体性能

4.5 避免指数项爆炸

传统高阶 FM 面临指数项爆炸问题。RFM 的解决方案：

关键突破： 通过复数旋转的角度累加特性，将高阶交互的计算复杂度从指数级 O(dᵏ) 降低到线性级 O(k)。
  • 角度累加： k 个特征的交互只需计算 k 个角度的和
  • 对数转换： 利用对数空间将乘法转换为加法
  • 近似计算： 对于极高阶交互，可以采用近似方法

5. 实验结果

5.1 实验设置

实验在多个公开 CTR 预测数据集上进行：

数据集 类型 样本数 特征数 稀疏性
Criteo 广告点击率 4500万 26 fields 高
Avazu 移动广告 4000万 23 fields 高
MovieLens-1M 电影推荐 100万 3 fields 中等

5.2 主要性能对比

AUC 指标（越高越好）：

模型 Criteo Avazu MovieLens
FM 0.7921 0.7732 0.7845
DeepFM 0.8089 0.7844 0.8043
xDeepFM 0.8102 0.7851 0.8056
EulerNet 0.8135 0.7882 0.8123
RFM (本文) 0.8148 0.7895 0.8138

结论： RFM 在所有数据集上都取得了最优性能，相比 EulerNet 进一步提升，验证了旋转因子分解机框架的有效性。

5.3 消融实验

作者进行了详细的消融实验，验证各组件的有效性：

  • 去除自注意力机制： AUC 下降约 0.3%，说明关系感知的重要性
  • 去除隐式交互（仅用显式旋转）： AUC 下降约 0.4%，说明两者互补
  • 去除极角表示（改用实数向量）： AUC 下降约 0.6%，说明复数表示的核心作用
  • 固定交互阶数： 性能下降且灵活性降低

5.4 计算效率对比

在 Criteo 数据集上比较训练时间：

  • FM：约 15 分钟
  • xDeepFM (3阶)：约 120 分钟
  • EulerNet：约 45 分钟
  • RFM：约 40 分钟（与 EulerNet 相当）

关键发现： 尽管 RFM 支持无界阶交互，但计算效率与 EulerNet 相当，远优于传统高阶方法。

6. 收益点

6.1 性能提升

  • 在多个标准 CTR 预测数据集上均取得 SOTA 性能
  • 相比 EulerNet 进一步提升 AUC 指标
  • 在不同稀疏度的数据集上都表现稳定

6.2 无界阶交互 核心优势： RFM 突破了传统 FM 的阶数限制，实现了真正意义上的无界阶特征交互，而计算复杂度仅为线性级。

  • 无需预定义最大交互阶数
  • 模型自动学习最优的交互阶数
  • 不同样本可以有不同的有效交互阶数

6.3 关系感知能力

  • 通过自注意力机制学习特征对之间的关系强度
  • 重要特征交互获得更高的权重
  • 噪声特征交互被自然抑制

6.4 计算效率

  • 复数旋转的计算开销很小
  • 线性复杂度支持大规模应用
  • 适合在线推荐系统的实时预测

6.5 可解释性

  • 旋转角度具有直观的物理意义
  • 注意力权重揭示了特征关系
  • 可以可视化特征在复数平面的分布

7. 局限性与未来工作

7.1 当前局限性(1) 模固定的限制：

  • RFM 将特征表示为单位复数（模为1），可能限制了表达能力
  • 某些场景下可能需要模可变的情况

(2) 注意力计算开销：

  • 自注意力机制在特征数量很多时计算复杂度较高
  • 对于超高维特征场景需要优化

(3) 序列数据适应性：

  • RFM 主要针对表格型特征设计
  • 对于序列特征的适应性有待验证

7.2 未来研究方向

(1) 扩展到序列推荐：

  • 将旋转机制与序列模型（如 Transformer）结合
  • 探索时序特征的旋转表示

(2) 多模态融合：

  • 将 RFM 应用于图像、文本等多模态特征
  • 探索跨模态的旋转交互机制

(3) 理论分析：

  • 深入分析旋转表示的表达能力上界
  • 研究不同旋转策略的理论性质

(4) 硬件优化：

  • 针对复数运算设计专用硬件加速
  • 优化大规模场景下的计算效率

8. 总结

Rotative Factorization Machines 是 KDD 2024 上的重要工作，它成功地将复数旋转机制引入经典的 Factorization Machines 框架，开创了"旋转因子分解"这一新的研究方向。

核心贡献回顾：

  1. 极角表示： 创新性地将特征表示为复数平面上的极角，实现了简洁而强大的特征编码
  2. 旋转交互： 通过复数旋转实现无界阶特征交互，突破了传统方法的阶数限制
  3. 关系感知： 引入自注意力机制，实现关系感知的特征交互建模
  4. 显隐统一： 巧妙地将显式旋转交互和隐式神经网络交互整合到统一框架
  5. 效率优化： 在保证表达能力的同时，维持了线性级的计算复杂度

RUCAIBox 复数网络研究线：

对 LLM 研究的启示：

RFM 对大语言模型研究也有重要启示：

  • 特征交互新范式： 旋转机制可以作为传统注意力机制的补充
  • 高效计算： 极角表示的线性复杂度对大规模模型有借鉴意义
  • 可解释性： 角度表示提供了新的模型可视化方法
总体评价： RFM 是一篇将经典机器学习模型与现代复数神经网络巧妙结合的典范工作。它不仅推动了因子分解机的发展，更为特征交互学习提供了全新的视角。RUCAIBox 团队在复数网络方向形成的系列工作（EulerNet → EulerFormer → RFM → π-Quant）展示了持续创新的研究能力，值得学术界和工业界关注。

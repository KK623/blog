🎯 核心思想

一句话总结：EulerFormer 是一个增强版的 RoPE（旋转位置编码），通过可学习的位置编码和语义自适应融合，在 Transformer 注意力机制中同时编码位置和语义信息。

核心公式

$$\\theta' = \\theta \\times \\delta + \\alpha \\; (+ b \\text{ for query})$$

θ (theta)
原始语义角度，从输入向量计算得到
δ (delta)
可学习的语义缩放因子
α (alpha)
可学习的位置编码角度
b (bias)
query 专用的旋转偏置

🔧 代码逐行解析

1. __init__ 方法 - 初始化参数

def __init__(self, config = None) -> None:
    super().__init__()                          # 调用父类 nn.Module 初始化
    self.alpha = None                           # 位置编码，延迟初始化
    
    if config.euler_bias:                       # 是否启用偏置
        self.b = nn.Parameter(torch.zeros(1))   # 可学习标量偏置
    else:
        self.b = 0                              # 无偏置
    
    self.delta = nn.Parameter(torch.ones(1) * config.init_factor)  
    # delta: 语义旋转缩放因子

    self.get_alpha(1, 1, config.MAX_ITEM_LIST_LENGTH, config.hidden_size)  
    # 预初始化位置编码

2. forward 方法 - 核心欧拉变换

def forward(self, v, type = 'ro'):
    v = v.unsqueeze(1)                          # [B, 1, seq, dim]
    
    # Step 1: 拆分为实部和虚部
    r = v[..., ::2]      # 偶数索引 → 实部
    p = v[..., 1::2]     # 奇数索引 → 虚部
    
    # Step 2: 直角坐标 → 极坐标
    lam = torch.sqrt(r ** 2 + p ** 2)   # 模长 λ = √(r²+p²)
    theta = torch.atan2(p, r)            # 相位角 θ = arctan(p/r)
    
    # Step 3: 旋转（核心）
    if 'ro' in type:
        alpha = self.get_alpha(...)       # 获取位置编码
        theta = theta * self.delta + alpha  # θ' = θ×δ + α
        if 'query' in type:
            theta = theta + self.b        # query 加偏置
    
    # Step 4: 极坐标 → 直角坐标
    r, p = lam * torch.cos(theta), lam * torch.sin(theta)
    
    # Step 5: 重新组合
    embeddings = torch.stack([r, p], dim=-1)
    return embeddings.squeeze(1)

3. get_alpha 方法 - 可学习位置编码

def get_alpha(self, batch_size, nums_head, max_len, output_dim):
    if self.alpha is None:                      # 延迟初始化
        position = torch.arange(0, max_len).unsqueeze(-1)  # [max_len, 1]
        ids = torch.arange(0, output_dim // 2)              # [dim//2]
        
        # RoPE 公式: θ_i = 10000^(-2i/d)
        theta = torch.pow(10000, -2 * ids / output_dim)
        
        embeddings = position * theta           # [max_len, dim//2]
        embeddings = torch.stack([embeddings, embeddings], dim=-1)
        
        self.alpha = nn.Parameter(embeddings)   # 设为可学习参数！
    
    # 扩展到 batch 和 head 维度
    embeddings = self.alpha.repeat((batch_size, nums_head, ...))
    return embeddings[...,::2]

  • 在 Q/K 投影后调用 EulerFormer

📊 与 RoPE 对比

🔵 RoPE

  • 位置编码：固定公式
  • 语义融合：无
  • Query偏置：无
  • 正则化：无
  • 理论：旋转位置编码

🟢 EulerFormer

  • 位置编码：可学习
  • 语义融合：delta缩放
  • Query偏置：有 b 参数
  • 正则化：对比学习损失
  • 理论：复数向量注意力

💡 总结

EulerFormer = RoPE的增强版 + 可学习位置编码 + 语义自适应融合 + 对比学习正则

核心创新

  1. 欧拉变换：直角坐标 ↔ 极坐标，仅旋转相位保持模长
  2. 可学习位置编码：alpha 是可学习参数，比 RoPE 更灵活
  3. delta 缩放：语义角度 × 可学习因子，动态平衡语义与位置
  4. query 偏置：给 query 额外旋转角度，增加表达能力
  5. 对比学习：辅助损失增强表示判别性

适用场景

  • 序列推荐（SASRec, BERT4Rec）
  • 长序列建模（LLaMA 改造）
  • 需要细粒度位置控制的场景

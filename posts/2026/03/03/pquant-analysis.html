pQuant 详细分析报告

2026-03-03 · 第 10 周

论文: arXiv: 2602.22592

作者: Wenzheng Zhang(北大), Bingzheng Liu(复旦), Yang Hu(军科院), Xiaoying Bai, Wentao Zhang(北大), Bin Cui(北大)

机构: 北京大学、复旦大学、军事科学院

1. 研究背景

1.1 LLM 部署的挑战

大语言模型(LLM)在各种NLP任务上取得了显著成绩，但它们巨大的计算和内存需求给实际部署带来了巨大挑战，尤其是在资源受限的设备上。量化(Quantization)通过降低模型权重和激活的数值精度，已成为缓解这些约束的关键技术。

1.2 极低比特量化的机遇

极低比特(sub 2-bit)量化因其能提供最激进的模型压缩而脱颖而出，可以大幅降低内存和计算足迹。其核心硬件优势在于用高效的位运算(bitwise operations)替代昂贵的浮点矩阵乘法，为边缘设备部署提供了有前途的路径。

1.3 现有方法的瓶颈

尽管极低比特量化取得了进展，但现有方法面临两个关键问题：

  1. 精度gap：即使是最先进的1-bit方法仍存在不可忽视的精度差距
  2. 扩展性差：随着模型增大，性能增益呈次线性增长，远落后于FP16模型

2. 核心发现：参数民主化效应

2.1 敏感性的定义

在量化理论中，参数具有不同的敏感性——少数"敏感"参数不成比例地影响模型输出。一个权重对量化的敏感性取决于：

  • 它是否远离量化网格点（较大的舍入误差）
  • 它是否经常与大的输入激活相乘（放大误差）

2.2 敏感性的数学定义

参考SPQR的方法，采用基于扰动的度量来分析极端低比特设置下的敏感性。对于权重矩阵W中的权重wij，定义敏感性为：

sij = min‖WX - W'X‖²

这衡量了量化该权重后输出失真的最小增加。

2.3 关键发现

论文发现了一个关键现象：

  • FP16模型：存在明显的不均匀敏感性分布，少量参数具有显著更高的敏感性
  • 1-bit BitNet模型：敏感性趋于均匀分布，呈现"扁平化"

论文将这种现象定义为参数民主化效应——在极端量化下，参数敏感性的意外同质化。

2.4 问题根源

参数民主化导致模型无法有效优先处理最有信息量的特征，这是限制极低比特LLM表达能力和扩展性的关键因素。

3. pQuant 方法

3.1 核心思想

pQuant通过结构化解耦参数来对抗参数民主化。其核心是将线性层拆分为两个专门分支：

  1. 主分支(1-bit)：保证基础效率
  2. 高精度分支：保留最敏感的参数

关键创新是不预先分配重要性，而是通过特征缩放(Feature Scaling)动态引导模型将最具影响力的表示分配到高精度路径。

3.2 1-bit + 8-bit 混合实现详解

3.2.1 架构总览

输入 → [特征缩放] → 1-bit分支 → 输出 (+)
            ↘ [路由器] → 8-bit分支 ↗

3.2.2 特征缩放 (Feature Scaling)

  • 对输入激活做归一化处理
  • 通过可学习的缩放因子调整特征分布
  • 引导模型动态学习：哪些特征需要高精度

3.2.3 双分支设计

  • 1-bit分支：大部分权重
    • 使用 Sign(w) 量化：将权重量化为 -1 或 +1
    • 极致的压缩率和计算效率
  • 8-bit分支：小部分权重
    • 使用 Absmax 量化：将权重保留为8位整数
    • 保留高精度以保存敏感参数

3.2.4 路由器 (Router)

  • 轻量级网络，为每个token选择激活哪个专家
  • 稀疏激活：每次只选一个专家，推理成本几乎不变
  • 类似MoE(Mixture of Experts)的设计

3.2.5 训练过程

  • 两个分支一起训练
  • 通过特征缩放引导敏感参数自然流向8-bit分支
  • 不需要预设哪些参数是高敏感的，让模型自己学会分配

3.3 具体应用

  • 多头注意力模块(MHA)：替换为纯1-bit线性层
  • 前馈网络模块(FFN)：使用解耦的1-bit + 8-bit双分支设计

4. 实验结果

4.1 困惑度对比(WikiText2, 1.3B模型)

方法 权重比特数 困惑度(PPL)
pQuant (本文) 1.35-bit 17.2
BitNet 1-bit 21.8
OmniQuant 2-bit 42.4
PTQ1.61 1.61-bit 39.6
BiLLM 1.11-bit 69.9

4.2 主要成果

  • 困惑度降低32.0%：相比最先进的1-bit基线
  • 超越2-bit模型精度：仅用1.35-bit
  • 推理吞吐量提升18.2%：相比2-bit模型
  • 匹配FP16性能：同时提供超过2倍的吞吐量

5. 创新点总结

  1. 首次发现参数民主化效应：揭示了极低比特量化的新瓶颈
  2. 解耦线性层设计：结构化分离效率与精度
  3. 特征缩放引导：动态分配敏感参数，非预设
  4. 稀疏专家扩展：近常数成本下扩容
  5. 显著超越SOTA：1.35-bit达到17.2困惑度

6. 意义与影响

pQuant为边缘设备提供了可扩展的极低比特量化方案，打开了1-bit LLM的大门。其方法证明了通过恢复参数的差异化敏感性结构，可以显著提升极低比特LLM的性能和扩展性。

1-bit + 8-bit混合的核心创新在于不是硬编码哪些权重是1-bit或8-bit，而是让模型通过训练自己学会把重要的特征分配到高精度分支。

LLM 量化与部署最新进展

2026-03-02 · 第 10 周

调研人: Subagent

摘要

本报告聚焦于2026年2月下旬至3月初发布的大语言模型（LLM）量化、部署、低精度及蒸馏相关最新论文。

一、量化技术最新进展

1.1 高效低比特量化

  • pQuant: 解耦线性量化感知训练，用于构建高效的低比特语言模型
  • NanoQuant: 亚1比特量化，突破1比特限制
  • TernaryLM: 原生1比特量化 + 自适应层缩放
  • RaBiT: 残差感知二值化训练，硬件友好的 matmul-free 推理
  • BPDQ: 位平面分解量化，灵活位宽

1.2 量化与推理优化

  • >QTALE: 量化鲁棒的 Token 自适应层执行
  • Astro: 激活引导结构化正则化，解决量化中的异常值问题
  • MoBiQuant: Token 自适应弹性 LLM 的混合位量化

    • 二、部署与推理优化

    2.1 边缘端部署

    • TOM: 三元只读存储器加速器，边缘设备部署
    • Gemma3 NPU 部署: AMD Ryzen AI NPU 上的端到端部署

    2.2 高效推理架构

    • 状态空间模型张量并行: 多 GPU 分布式推理
    • SpeContext: 长上下文推理的投机上下文稀疏化

    三、模型压缩与蒸馏

    • UniComp: LLM 压缩的统一评估框架
    • GradPruner: 梯度引导的层剪枝
    • SlimMoE: MoE 模型的结构化压缩

    四、总结与趋势

    当前研究热点：
    1. 超低比特量化 (1-bit、2-bit)
    2. Token 自适应量化
    3. 边缘端部署 + 硬件协同
    4. 量化鲁棒性研究
    5. 多模态模型量化

    技术路线对比

    方向代表工作优势
    1-bit 量化TernaryLM, RaBiT极致压缩
    混合位量化MoBiQuant灵活性高
    后训练量化D²Quant, MatGPTQ无需再训练
    硬件部署TOM, Gemma3 NPU实际可用

    下一步方向建议

    • 关注 1-bit 量化在实际部署中的可行性
    • Token 级量化与推理系统的深度整合
    • MoE 模型的压缩与部署
    • 长上下文场景下的效率优化

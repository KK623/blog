开源工具与框架

复数神经网络(CVNN)生态系统日益丰富，以下整理了当前主流的8个开源框架，按PyTorch、TensorFlow及其他工具进行分类。

PyTorch 生态系统

推荐

torchcvnn

Victor Dhédin, Jérémie Levi (CentraleSupélec)

  • •基于PyTorch后端的复数神经网络库
  • •提供复数值数据集（遥感、MRI）
  • •支持复数卷积、批量归一化等
  • •完整的复数值变换和层实现
pip install torchcvnn
查看文档

complexPyTorch

wavefrontshaping团队

  • •高级PyTorch复数神经网络工具箱
  • •支持PyTorch 1.7+的complex64张量
  • •提供复数卷积、池化、归一化层
  • •支持Complex BatchNorm（Naive/Covariance）
pip install complexPyTorch

引用: PhysRevX.11.021060

complextorch

Josiah W. Smith, Ph.D.

  • •轻量级PyTorch复数神经网络包
  • •专注信号处理、通信和雷达数据
  • •高效实现线性、卷积和注意力模块
  • •支持1D/2D/3D数据
pip install complextorch

文档: Read the Docs + arXiv论文

cplxmodule

Ivan Nazarov等研究者

  • •PyTorch扩展，支持复数层和激活函数
  • •实现实数和复数的变分dropout
  • •支持网络稀疏化和剪枝
  • •基于Wirtinger微积分
pip install cplxmodule

依赖: numpy, torch, scipy

框架选择建议

PyTorch新项目

推荐使用 torchcvnn 或 complexPyTorch，两者均维护活跃、文档完善。

研究用途

如需网络稀疏化功能，选择 cplxmodule，支持贝叶斯剪枝和变分dropout。

TensorFlow遗留项目

使用 cvnn 时需注意版本兼容性（TF 2.16+不兼容）。

CVNN-01 复数神经网络基础概念详解

生成时间: 2026-03-06 17:55 | 文章编号: CVNN-01 | 预计阅读: 15分钟

1. 为什么需要复数神经网络？

传统的神经网络使用实数进行计算，但在许多实际应用中，数据天然具有复数形式：

  • 信号处理：傅里叶变换后的频域信号是复数
  • 雷达/SAR：回波信号包含幅度和相位信息
  • 光学：光波的振幅和相位
  • 量子计算：量子态是复数向量
核心优势：复数神经网络可以同时处理幅度和相位信息，在频域数据处理上比实数网络更高效。

2. 复数基础回顾

2.1 复数表示

复数 z 可以表示为：

z = a + bi = r·e^(iθ)

其中：

  • a = Re(z) 是实部
  • b = Im(z) 是虚部
  • r = |z| = √(a² + b²) 是幅度
  • θ = arctan(b/a) 是相位

2.2 复数运算

加法：

(a + bi) + (c + di) = (a+c) + (b+d)i

乘法：

(a + bi) × (c + di) = (ac - bd) + (ad + bc)i

共轭：

z* = a - bi

3. 复数神经元

复数神经元的输入、权重和输出都是复数：

z_out = σ(W · z_in + b)

其中：

  • z_in ∈ ℂⁿ 是复数输入向量
  • W ∈ ℂ^{m×n} 是复数权重矩阵
  • b ∈ ℂᵐ 是复数偏置
  • σ 是复数激活函数

4. 实部-虚部分解视角

复数运算可以分解为实数运算：

[Re(z_out)] [Re(W) -Im(W)] [Re(z_in)] [Re(b)] [Im(z_out)] = [Im(W) Re(W)] × [Im(z_in)] + [Im(b)]
理解：一个复数神经元等价于一个2×2的实数权重矩阵，但参数量只有一半！

5. CVNN vs RVNN 对比

特性 实数神经网络 (RVNN) 复数神经网络 (CVNN)
输入 仅实数 复数 (幅度+相位)
参数效率 需要2×参数处理复数 直接处理，参数减半
频域处理 需要分解实部/虚部 天然适合

CVNN-03 复数神经网络激活函数详解

生成时间: 2026-03-06 17:57 | 文章编号: CVNN-03 | 预计阅读: 15分钟

1. 复数激活函数的挑战

实数激活函数（如ReLU、sigmoid）不能直接用于复数，因为：

  • 复数没有自然的大小顺序（无法比较 z₁ > z₂）
  • 直接对实部和虚部分开应用ReLU会破坏相位信息
  • 需要保持全纯性或合理的非线性特性

2. ModReLU (模值ReLU)

ModReLU是目前最流行的复数激活函数之一。

2.1 定义

ModReLU(z) = (|z| + b)₊ · (z/|z|)

其中：

  • |z| 是复数的幅度
  • b 是可学习的偏置参数
  • (x)₊ = max(0, x) 是ReLU函数
  • z/|z| 保持相位信息
直观理解：ModReLU对幅度进行阈值处理（类似ReLU），但保持相位不变。

2.2 特性

  • 相位保持：输出与输入具有相同相位
  • 幅度非线性：对幅度进行ReLU-like变换
  • 可学习阈值：通过b参数自适应调整

3. CReLU (复数ReLU)

3.1 定义

CReLU(z) = ReLU(Re(z)) + i·ReLU(Im(z))

即对实部和虚部分别应用ReLU。

3.2 问题与改进

CReLU的问题是可能改变相位。改进版本zReLU：

zReLU(z) = z if Re(z) > 0 and Im(z) > 0 0 otherwise

4. 复数 sigmoid 和 tanh

4.1 复数 sigmoid

σ(z) = 1 / (1 + exp(-z))

4.2 复数 tanh

tanh(z) = (exp(z) - exp(-z)) / (exp(z) + exp(-z))

注意：这些函数在复数域上是周期性的，可能导致梯度爆炸。

5. 相位敏感激活函数

5.1 Phased ReLU

PhasedReLU(z) = ReLU(|z| - b) · exp(i·θ)

其中 z = |z|·exp(i·θ)。

6. 激活函数选择指南

激活函数 适用场景 注意事项
ModReLU 通用首选 需要学习偏置b
CReLU 简单快速 可能改变相位
zReLU 保持象限信息 可能过于稀疏

7. 实现代码示例

# PyTorch 伪代码 def mod_relu(z, b): magnitude = torch.abs(z) phase = z / (magnitude + 1e-8) return F.relu(magnitude + b) * phase

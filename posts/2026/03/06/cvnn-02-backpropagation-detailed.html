CVNN-02 复数神经网络反向传播与优化详解

1. 复数梯度下降的挑战

复数函数 f(z) 的优化面临一个根本问题：复数域不是有序域，无法直接比较 "z₁ > z₂"。

核心问题：如何定义复数函数的导数和梯度？

2. Wirtinger 微积分

Wirtinger微积分提供了一种优雅的处理方式，将复数函数视为两个独立变量 z 和 z* 的函数。

2.1 Wirtinger 导数定义

对于复数函数 f(z)，定义两个偏导数：

∂f/∂z = ½(∂f/∂a - i·∂f/∂b) ∂f/∂z* = ½(∂f/∂a + i·∂f/∂b)

其中 z = a + bi，z* = a - bi。

2.2 重要性质

  • 如果 f 是全纯函数（满足柯西-黎曼条件），则 ∂f/∂z* = 0
  • 对于实值损失函数 L，∂L/∂z* 是共轭梯度，用于梯度下降

3. 复数梯度下降

3.1 梯度更新规则

复数参数的更新规则：

z ← z - η · ∂L/∂z*

其中：

  • η 是学习率
  • ∂L/∂z* 是损失函数对 z 的共轭梯度
关键理解：复数梯度下降沿着共轭梯度的负方向更新，这与实数梯度下降类似，但考虑了复数的特殊结构。

3.2 实部-虚部分解形式

复数梯度下降可以分解为实部和虚部的联合更新：

[Re(z)] [Re(∂L/∂z)] [Im(z)] ← [Im(z)] - η · [Im(∂L/∂z)]

4. CVNN 反向传播算法

4.1 前向传播

zₗ = Wₗ · aₗ₋₁ + bₗ aₗ = σ(zₗ)

4.2 反向传播

输出层梯度：

δₗ = ∂L/∂zₗ* = (∂L/∂aₗ*) ⊙ σ'(zₗ*)

隐藏层梯度：

δₗ = (Wₗ₊₁ᴴ · δₗ₊₁) ⊙ σ'(zₗ*)

其中 Wᴴ 表示共轭转置（Hermitian）。

4.3 参数梯度

∂L/∂Wₗ* = δₗ · aₗ₋₁ᴴ ∂L/∂bₗ* = δₗ

5. 复数优化器

5.1 复数 Adam

复数版本的Adam优化器：

mₜ = β₁·mₜ₋₁ + (1-β₁)·gₜ vₜ = β₂·vₜ₋₁ + (1-β₂)·|gₜ|² m̂ₜ = mₜ/(1-β₁ᵗ) v̂ₜ = vₜ/(1-β₂ᵗ) θₜ = θₜ₋₁ - η·m̂ₜ/(√v̂ₜ + ε)

其中 gₜ = ∂L/∂θ* 是共轭梯度。

6. 实现注意事项

  • 数值稳定性：复数除法需要处理分母接近零的情况
  • 初始化：复数权重的初始化需要考虑幅度和相位的分布
  • 正则化：L2正则化应作用于 |w|² = w·w*

CVNN-04 复数卷积层详解

生成时间: 2026-03-06 17:58 | 文章编号: CVNN-04 | 预计阅读: 18分钟

1. 复数卷积的定义

复数卷积层是CVNN的核心组件，特别适用于处理频域数据和具有相位信息的信号。

1.1 数学定义

对于输入特征图 Z ∈ ℂ^{H×W×C_in} 和卷积核 K ∈ ℂ^{k×k×C_in×C_out}：

Y[m,n] = Σᵢ Σⱏ K[i,j] · Z[m+i, n+j]

其中所有运算都是复数乘法。

1.2 实部-虚部分解

复数卷积可以分解为四个实数卷积：

Y_real = K_real * Z_real - K_imag * Z_imag Y_imag = K_real * Z_imag + K_imag * Z_real
关键洞察：一个复数卷积层等价于两个并行的实数卷积，但参数量相同！

2. 频域特性

复数卷积在频域具有优雅的形式：

F(Y) = F(K) ⊙ F(Z)

其中 F 表示傅里叶变换，⊙ 表示逐元素乘法。

这意味着：

  • 时域卷积等价于频域逐元素乘法
  • 复数卷积核可以同时学习幅度响应和相位响应

3. 复数卷积核初始化

3.1 幅度初始化

使用He初始化或Xavier初始化：

|K| ~ N(0, 2/(k²·C_in))

3.2 相位初始化

相位可以均匀随机初始化：

arg(K) ~ U(-π, π)

4. 复数批归一化

复数批归一化需要考虑实部和虚部的协方差：

Z_norm = (Z - μ) / σ

其中：

  • μ = E[Z] 是复数均值
  • σ² = E[|Z - μ|²] 是复数方差

5. 应用场景

  • SAR图像处理：雷达回波的相位信息至关重要
  • MRI重建：k空间数据是复数
  • 无线通信：信道估计和均衡
  • 光学成像：全息图像处理

6. 实现代码

# PyTorch 伪代码 class ComplexConv2d(nn.Module): def __init__(self, in_channels, out_channels, kernel_size): self.real_conv = nn.Conv2d(in_channels, out_channels, kernel_size) self.imag_conv = nn.Conv2d(in_channels, out_channels, kernel_size) def forward(self, z_real, z_imag): out_real = self.real_conv(z_real) - self.imag_conv(z_imag) out_imag = self.real_conv(z_imag) + self.imag_conv(z_real) return out_real, out_imag

复数神经网络在医学成像（MRI）中的应用

MRI数据天然具有复值特性（实部和虚部），复数神经网络能够更好地保留相位信息， 在MRI重建、指纹识别和数据生成等任务中展现出显著优势。

ICML 2017 论文 #1

Better than Real: Complex-valued Neural Nets for MRI Fingerprinting

arXiv:1707.00070
作者: Patrick Virtue, Stella X. Yu, Michael Lustig
机构: UC Berkeley（加州大学伯克利分校）

研究背景

MRI指纹识别是一种新兴的定量MRI技术，通过采集具有独特信号演变特征的"指纹"数据， 可以同时估计多种组织参数（T1、T2等）。传统方法需要建立庞大的字典进行模式匹配， 计算成本高昂且需要大量的存储空间。

核心动机

MRI信号本质上是复值信号，包含幅度和相位信息。传统实值神经网络将复数数据分离为两个实数通道处理， 无法有效捕获实部与虚部之间的内在关联。如何利用复数运算直接处理MRI信号成为关键问题。

技术创新

  • • 提出首个专门用于MRI指纹识别的复数神经网络架构
  • • 设计了创新的心形激活函数（Cardioid Activation），更适合复值数据处理
  • • 使用MRI模拟器生成大规模合成训练数据，实现端到端学习
  • • 网络直接将MRI信号映射到组织参数，无需字典匹配

收益与成果

  • ✓ 相比实值神经网络，准确率显著提升
  • ✓ 推理速度极快，从传统方法的数小时缩短到毫秒级
  • ✓ 大幅降低存储需求，无需维护庞大的指纹字典
  • ✓ 为后续MRI+深度学习研究奠定了重要基础

论文总结

这是CVNN在医学成像领域的里程碑式工作，首次证明了复数神经网络在MRI指纹识别任务中相比实值网络的显著优势。 论文提出的"心形激活函数"成为后续CVNN研究的重要参考。该工作开创了使用深度学习方法替代传统字典匹配的新范式， 对定量MRI的临床转化产生了深远影响。

2025 论文 #2

PhaseGen: A Diffusion-Based Approach for Complex-Valued MRI Data Generation

arXiv:2504.07560
作者: Moritz Rempe, Fabian Hörst, Helmut Becker, Marco Schlimbach, Lukas Rotkopf, Kevin Kröninger, Jens Kleesiek
机构: RWTH Aachen University（德国亚琛工业大学）

研究背景

深度学习模型通常需要大量训练数据，但获取带标注的医学影像数据困难且昂贵。 此外，许多先进的MRI分析方法（如k空间处理）需要复值原始数据，而临床常用的是幅度图像。 如何基于现有幅度图像生成对应的相位信息成为关键挑战。

核心动机

扩散模型在图像生成领域取得了巨大成功，但将其应用于复值MRI数据生成仍具挑战。 目标是以临床幅度图像为条件，生成物理合理的复值k空间数据，从而扩充训练数据集， 并支持需要复值输入的下游任务。

技术创新

  • • 首个基于扩散模型的复值MRI数据生成框架
  • • 以幅度图像为条件生成合成相位，实现k空间数据增强
  • • 设计了适合复值数据的扩散过程和噪声调度策略
  • • 支持需要k空间信息的模型进行预训练

收益与成果

  • ✓ 颅骨剥离泛化能力大幅提升：分割准确率从41.1%提升到80.1%
  • ✓ 在MRI重建任务上验证有效
  • ✓ 解决了复值模型训练数据稀缺的难题
  • ✓ 为医学影像数据增强开辟了新方向

论文总结

PhaseGen将扩散模型成功应用于复值MRI数据生成领域，是2025年的前沿工作。该方法巧妙地利用临床常用的幅度图像作为条件， 生成对应的相位信息，从而合成完整的复值k空间数据。实验结果表明，使用合成数据预训练可以显著提升模型在真实数据上的泛化能力， 为解决医学影像领域的数据稀缺问题提供了创新思路。

ISMRM 2019 论文 #3

DeepcomplexMRI: Exploiting Deep Residual Network for Fast Parallel MR Imaging with Complex Convolution

arXiv:1906.04359
作者: Shanshan Wang, Huitao Cheng, Leslie Ying, Taohui Xiao, Ziwen Ke, Xin Liu, Hairong Zheng, Dong Liang
机构: 中国科学院深圳先进技术研究院

研究背景

并行成像（Parallel Imaging）通过多线圈采集加速MRI扫描，但传统方法如GRAPPA、SENSE在高度加速时会产生显著伪影。 近年来，基于深度学习的方法展现出潜力，但早期方法多采用双通道实值网络分别处理实部和虚部，忽略了它们之间的相关性。

核心动机

MR图像本质上是复值的，其实部和虚部通过物理约束相互关联。传统的双通道实值方法将复数数据视为两个独立的实数通道， 破坏了这种内在联系。如何设计能够直接处理复值数据并维护k空间数据一致性的网络架构成为关键问题。

技术创新

  • • 提出DeepcomplexMRI：基于残差复数卷积神经网络的快速并行MRI重建方法
  • • 设计了复数卷积层，能够显式建模实部和虚部之间的相关性
  • • 在网络层之间强制执行k空间数据一致性，确保重建结果物理可行
  • • 利用多线圈数据离线训练深度残差网络

收益与成果

  • ✓ 与当时最先进的方法相比，重建精度显著提升
  • ✓ 能够恢复高质量的多通道图像
  • ✓ 验证了复数卷积在MRI重建中的有效性
  • ✓ 为后续压缩感知MRI重建研究提供了重要参考

论文总结

DeepcomplexMRI是CVNN在MRI重建领域的开创性工作之一，首次将复数卷积应用于快速并行MRI成像。 论文的创新之处在于不仅使用了复数卷积层，还在网络结构中引入了k空间数据一致性约束， 确保重建结果满足MRI物理模型。这项工作为后续Co-VeGAN、CV-CNN等研究奠定了基础， 展示了复数神经网络在医学成像领域的巨大潜力。

2021 论文 #4

Complex-valued Federated Learning with Differential Privacy and MRI Applications

arXiv:2110.03478
作者: Anneliese Riess, Alexander Ziller, Stefan Kolek, Daniel Rueckert, Julia Schnabel, Georgios Kaissis
机构: Technical University of Munich（德国慕尼黑工业大学）

研究背景

联邦学习（Federated Learning）允许多个机构在不共享原始数据的情况下协作训练模型， 对于保护患者隐私的医学影像研究具有重要意义。差分隐私（Differential Privacy） 提供了严格的数学框架来量化隐私保护程度。然而，将这两种技术应用于复数神经网络尚未被探索。

核心动机

医学数据天然分布在多个医疗机构中，且隐私法规严格。如何在保护患者隐私的前提下， 利用分布式数据训练复数神经网络处理k空间MRI数据？需要扩展现有的差分隐私理论到复数域。

技术创新

  • • 首次引入复数高斯机制，在f-DP、(ε,δ)-DP和Rényi-DP框架下表征其行为
  • • 将DP-SGD算法推广到复数神经网络
  • • 提出与差分隐私兼容的新型复数神经网络原语
  • • 实现了联邦学习场景下的隐私保护复数模型训练

收益与成果

  • ✓ 在MRI脉冲序列分类任务上验证了概念可行性
  • ✓ 在提供强隐私保证的同时保持了模型实用性
  • ✓ 为跨机构协作的医学影像AI研究开辟了道路
  • ✓ 建立了复数域差分隐私的理论基础

论文总结

这是首个将联邦学习和差分隐私引入复数神经网络的研究工作，在理论和实践上都具有重要意义。 论文从理论上建立了复数高斯机制的隐私分析框架，并在真实的MRI分类任务上进行了验证。 这项工作为医学影像领域的隐私保护AI研究开辟了新方向，特别是在需要跨机构协作的场景下， 如多中心临床研究中的MRI数据共享与分析。

章节小结

复数神经网络在MRI应用中展现出独特优势。上述四篇论文分别覆盖了CVNN在医学成像中的核心应用场景： MRI指纹识别（Virtue et al., 2017）开创了深度学习替代字典匹配的新范式； MRI数据生成（Rempe et al., 2025）利用扩散模型解决复值训练数据稀缺问题； MRI重建（Wang et al., 2019）通过复数卷积和k空间一致性约束提升成像质量； 联邦学习与隐私保护（Riess et al., 2021）则解决了跨机构协作的隐私难题。

这些研究共同证明了：对于MRI这类固有复值信号，采用复数神经网络相比传统双通道实值方法能够更好地保留相位信息、 建模实部虚部关联，从而在精度、效率和泛化能力等方面取得显著改进。
